# Кейс-задача №5

## Анализ кейс-задачи №4 (интерактивный калькулятор)

Для анализа я выбрал кейс-задачу №4 — создание калькулятора на HTML, CSS и JavaScript.

## Функциональность

Калькулятор выполняет основные функции: сложение, вычитание, умножение и деление.  
Пользователь вводит два числа и нажимает нужную кнопку, после чего результат выводится на экран.

Также реализована проверка ввода — если пользователь вводит не числа, появляется сообщение об ошибке.  
Добавлена проверка деления на ноль.

## Производительность

Программа работает быстро, так как не использует сложные вычисления и сторонние библиотеки.  
Все операции выполняются мгновенно после нажатия кнопки.

## Удобство использования

Интерфейс простой и понятный: есть два поля ввода, кнопки операций и окно для результата.  
Пользователь сразу понимает, что делать.

Ошибки выводятся в текстовом виде, что помогает понять проблему.

## Безопасность

Так как это клиентское приложение без сервера и базы данных, риски минимальные.  
Для вывода результата используется безопасный способ (без вставки HTML-кода).

## Масштабируемость

Программу можно расширять — например:
- добавить новые операции
- добавить историю вычислений
- сделать более сложный калькулятор

Но при увеличении функционала код нужно будет немного переработать.

## Сопровождаемость

Код разделён на файлы:
- HTML — структура
- CSS — стили
- JavaScript — логика

Это удобно, так как можно отдельно менять внешний вид и функционал.

## Переносимость

Программа работает в любом браузере и не требует установки.  
Достаточно открыть файл index.html.

## Качество кода

Код написан достаточно понятно, переменные названы логично.  
Структура простая, без лишней сложности.

## Тестирование

Были проверены разные варианты:
- обычные числа
- отрицательные числа
- дробные числа
- ввод текста
- пустые поля
- деление на ноль

Во всех случаях программа либо считает правильно, либо показывает ошибку.

## Возможные улучшения

Можно улучшить программу:
- добавить кнопку очистки
- сделать красивый вывод ошибок
- добавить округление результата
- улучшить дизайн
- адаптировать под телефон

## Вывод

Калькулятор выполняет свою задачу и соответствует требованиям задания.  
Программа простая, понятная и работает корректно.  
При желании её можно легко доработать и расширить.
